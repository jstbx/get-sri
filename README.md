[![JavaScript Style Guide][jsstd-badge]][jsstd-url]

A helper tool to **keep package.json lean** by extracting 
**your NPM scripts into a separate file**.

## Installation

```bash
# Using NPM
npm install get-sri --save-dev
```

```bash
# Using Yarn
yarn add get-sri --dev
```
